## 1. 帳票出力（#59）
取引詳細（`/transactions/[id]`）→「帳票（請求書・明細・証明書）」セクション。
- [ ] **取引明細書**：常に「開く」で新タブ表示。案件名・当事者・工期・各フェーズの作業/請求状況・合計が正しい
- [ ] **請求書**：請求書提出**前**は「未発行」表示 → 協力会社が請求書を提出**後**に「開く」が有効
  - [ ] 請求元＝協力会社／宛先＝元請／金額・支払期日・振込先が正しい
- [ ] **取引完了証明書**：取引完了（入金確認まで）**前**は「未発行」→ 完了**後**に発行可
  - [ ] 完了日・取引金額・期日内/遅延の別が正しい
- [ ] 各帳票ページ下部の「🖨 印刷 / PDFに保存」でブラウザ印刷 → PDF 保存できる
- [ ] 印刷プレビューでボタン・戻るリンクが消える（帳票のみ印刷される）
- [ ] **第三者拒否**：当事者でない会社／未ログインで `/print/tx/<他人の取引ID>/statement` を開くと「表示できません」案内（中身は見えない）

## 4. PWA インストール＆プッシュ通知（#63）
### インストール
- [ ] iPhone/Safari：本番URLを開き 共有 →「ホーム画面に追加」→ アイコンから**スタンドアロン起動**（アドレスバーなし）
- [ ] Android/Chrome：インストール案内 or メニューから「アプリをインストール」
### 購読
- [ ] マイアカウント →「プッシュ通知」→「通知をオンにする」→ 許可 → オンになる
- [ ] 「テスト通知」→ 数秒以内に端末へ通知が届く
- [ ] 通知タップでアプリが開く（該当画面へ遷移）
- [ ] 「オフにする」で解除できる
### イベント通知（2端末 or 2アカウントで確認）
- [ ] 元請の案件に別会社が**応募** → 元請端末へ「新しい応募」通知
- [ ] 元請が協力会社を**選定** → 選定された会社の端末へ「選定されました」通知
- [ ] **チャット**送信 → 相手側の端末へ「新着メッセージ」通知（本文プレビュー付き）
- [ ] アプリを閉じている/バックグラウンドでも通知が届く
### graceful degradation
- [ ] VAPID 未設定のうちは「未設定」と表示され、画面が壊れない（設定後に有効化）
- [ ] 非対応ブラウザ（iOS でホーム追加前の Safari など）で「対応していません」案内
